
< FrontEnd-設計 >

[ 概要 ]

管理者が動的にカスタマイズできるアンケートForm-パッケージの開発

[ FrontEndの使用技術 ]

  1. React

  2. Redux => 状態管理

[ ページの定義 ]

  1. ご要望アンケート・フォーム(Topページ)

  2. 入力内容の確認画面(Confirmページ)

  3. 受付完了(Thanksページ)


[ 機能の定義 ]

  1. ページ・タイムスライダー

  2. 必須・任意の動的な作成

  3. 入力フォームの動的な作成

  4. 必須の入力フォームは、入力しているかCheckする

  5. 検索機能付きのSelectBoxを作成する

  6. 確認画面に進むボタンで、確認画面に進める

  7. 確認画面では、入力画面に戻るボタンと送信ボタンがある
